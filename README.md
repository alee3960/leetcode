# Project Description

# Prerequisites
1. Python
2. UV

# Instructions
1. Clone the repo:
```bash 
git clone https://github.com/alee3960/leetcode.git
```
2. Navigate to the directory and problem
```bash
cd leetcode
cd '.\9. Palindrome Number\'
```
3. Run the file or tests for the solution
```bash
uv run solution.py
uv run pytest
```
